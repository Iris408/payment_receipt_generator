# Payment Receipt Generator

A beginner Python project that generates a PDF payment receipt using ReportLab.

## Version 1.0

This version creates a static payment receipt with:

- Receipt title
- Customer name
- Receipt number
- Receipt date
- Item table
- Sample PDF output

## Tech Stack

- Python
- ReportLab
- PDF generation
